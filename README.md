# HH Job Tool

Desktop-приложение для работы с [hh.ru](https://hh.ru): авторизация, расширенный поиск вакансий, автоотклики с тонкой настройкой, просмотр откликов и краулер `data-qa` элементов сайта.

Стек: **Electron** + **Playwright** (stealth-режим).

![License](https://img.shields.io/badge/license-ISC-blue)

## Возможности

| Раздел | Описание |
|--------|----------|
| **Авторизация** | Вход по телефону/email и паролю, поддержка SMS-кода (OTP) |
| **Поиск** | Мультистраничный сбор (до 10 страниц), все серверные фильтры hh.ru (формат работы, занятость, график, часы, образование, метки, инклюзивность, период, сортировка) + клиентские фильтры (зарплатный диапазон, рейтинг, отзывы, исключения слов), сортировка, подсветка совпадений, сохранённые поиски |
| **Автоотклик** | Массовые отклики с расширенными фильтрами (white-list, режим ключевых слов И/ИЛИ, мин. рейтинг/отзывы), несколько шаблонов писем с переменными `{title}`/`{employer}`/`{salary}`, dry-run, retry, расписание по интервалу |
| **Отклики** | Загрузка переговоров с работодателями, статистика по статусам, фильтры, поиск, сортировка |
| **Краулер** | Сбор `data-qa` селекторов со страниц hh.ru для анализа вёрстки |
| **CLI** | Скрипты `init.js` и `parse.js` для авторизации и парсинга без GUI |

## Требования

- **Node.js** 18+ (для запуска из исходников)
- **Windows / macOS / Linux**
- Аккаунт соискателя на hh.ru

## Установка из исходников

```bash
git clone https://github.com/lonestill/hruhru.git
cd hruhru
npm install
npm start
```

Chromium для Playwright скачается при первом запуске — будет показан прогресс-бар. Это разовая операция (~150MB).

## Готовые сборки

На странице [Releases](https://github.com/lonestill/hruhru/releases) доступны готовые билды под Windows, macOS и Linux. Они обновляются автоматически при каждом пуше в `main`.

При первом запуске готового билда приложение само скачает Chromium — это нужно, чтобы установщик оставался лёгким.

## Запуск

### GUI (основной режим)

```bash
npm start
```

При первом запуске откроется экран входа. Введите телефон (10 цифр без +7) или email — откроется окно браузера для авторизации на hh.ru. При запросе SMS-код вводится в приложении.

### CLI

**Авторизация** (сохраняет сессию в `auth.json`):

```bash
npm run cli:auth
```

Можно передать учётные данные через переменные окружения:

```bash
# Windows PowerShell
$env:HH_LOGIN="9000000000"; $env:HH_PASSWORD="your_password"; npm run cli:auth

# Linux / macOS
HH_LOGIN=9000000000 HH_PASSWORD=your_password npm run cli:auth
```

**Краулер** (требует `auth.json`):

```bash
npm run cli:parse
```

## Локальные данные

Эти файлы создаются при работе и **не коммитятся** в git:

| Файл | Назначение |
|------|------------|
| `auth.json` | Сессия Playwright (cookies) |
| `settings.json` | Настройки приложения |
| `crawl_state.json` | Состояние очереди краулера |
| `hh_global_elements_map.json` | Результаты краулера |

Примеры-шаблоны: `auth.json.example`, `settings.json.example`, `crawl_state.json.example`.

## Скрипты npm

| Команда | Описание |
|---------|----------|
| `npm start` | Запуск Electron GUI |
| `npm run dev` | GUI с `--inspect` для отладки |
| `npm run cli:auth` | CLI-авторизация |
| `npm run cli:parse` | CLI-краулер |
| `npm run check` | Проверка синтаксиса JS-файлов |
| `npm run dist` | Локальная сборка под текущую ОС |
| `npm run dist:win` | Сборка под Windows (nsis) |
| `npm run dist:mac` | Сборка под macOS (dmg, x64 + arm64) |
| `npm run dist:linux` | Сборка под Linux (AppImage + deb) |

## CI/CD

**CI** (`.github/workflows/ci.yml`) — на каждый push/PR: проверка синтаксиса через `npm run check`.

**Release** (`.github/workflows/release.yml`) — на push в `main`:
1. Вычисляет версию в формате `vYYYY.MM.DD-<sha7>` (например, `v2026.07.13-abc1234`)
2. Собирает билды на Windows, macOS и Linux параллельно
3. Создаёт git-тег и GitHub Release с загруженными артефактами

## Безопасность

- **Никогда** не коммитьте `auth.json`, `.env` и другие файлы с сессией или паролями.
- Учётные данные вводятся в GUI или через переменные окружения / интерактивный ввод в CLI.
- Если пароль когда-либо попадал в историю git — **смените его** на hh.ru.

## Ограничения и дисклеймер

- Проект использует автоматизацию браузера и не является официальным клиентом hh.ru.
- Соблюдайте [правила использования сайта](https://hh.ru/article/28) и не злоупотребляйте автооткликами.
- hh.ru может показывать капчу или блокировать подозрительную активность.
- Используйте на свой страх и риск.
